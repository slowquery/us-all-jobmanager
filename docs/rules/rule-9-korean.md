# Rule 9 — 한글 커뮤니케이션

## 요약
사용자 대상 제안/권고, **PR 제목**, **커밋 메시지 내용**은 한글로 작성한다. 코드/식별자는 기존 컨벤션을 따른다(타입 프리픽스만 영문).
거버넌스 스크립트(`.githooks/*`·`scripts/*`)·CI의 **사용자 노출 출력 문구**와 앱의 사용자 대면 메시지도 한글로 작성한다.
**결정 로그(`logs/**/*.md`) 본문도 한글**이어야 한다(Rule 4).

## 강제
제안 프로즈는 어드바이저리; 커밋 메시지 한글은 하드(`commit-msg` 훅); 결정 로그 본문 한글은 하드(CI `결정 로그 한글 게이트`).
