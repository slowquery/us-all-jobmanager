# Rule 8 — AGENTS.md / CLAUDE.md

## 요약
루트 `AGENTS.md`는 GJC가 주입하는 **목차(TOC)**이고, 상세는 `docs/*`로 분리해 필요 시 참조한다. `CLAUDE.md`는 `@AGENTS.md`로 미러(무드리프트). `.gjc/AGENTS.md`는 루트를 섀도잉하므로 **생성 금지**.

## 강제
어드바이저리(구조적으로 검증). 정규는 `.gjc/rules/*`.

## 참고
이 규칙(문서 계층)은 자기참조적이라 별도 `.gjc/rules` 파일 없이 AGENTS.md/docs에서 정규다.
